int PS_2_3::set ( double const &  rs,
double const &  y_cm,
double const &  phi_cm,
double const &  M12,
double const &  y_12,
double const &  phi_12 
)

Compute new phase space setting: The 2->3 phase space is split into p1 p2 -> Q k3 and Q -> k1 k2. y_cm is the angle of k3 in the p1 + p2 c.m.f., the mass of the intermediate state Q is M12 = sqrt((k1+k2)^2), the angles y_12 and phi_12 define the direction of k1' and k2' defined in the Q-restframe.

Parameters
rs
y_cmazimuthal scattering angle of k3 in p1+p2 z.m.f. (z-axis is the reference axis)
phi_cmpolar scattering angle of k3 in p1+p2 z.m.f. (z-axis is the reference axis)
M12invariant mass of the system k1+k2
y_12azimuthal scattering angle of k2 in k1+k2 z.m.f. (z-axis is the reference axis)
phi_12polar scattering angle of k2 in k1+k2 z.m.f. (z-axis is the reference axis)
