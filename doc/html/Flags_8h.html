Detailed Description

Flags to control the evaluation of sub amplitudes.

Note that the flags that select subamplitudes are only active when compiled with DEBUG=1. Default is DEBUG=0, where all NLO subamplitudes in the files Functions_pp_ttX_R/V/ID/UID.cpp get evaluated.

See also
Functions_pp_ttX_V.cpp, Functions_pp_ttX_ID.cpp, Functions_pp_ttX_R.cpp, Functions_pp_ttX_UID.cpp
