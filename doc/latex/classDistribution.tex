\subsection{Detailed Description}
Class for differential distributions. 

The distribution class holds a pointer to a storage object (\hyperlink{classHistArray}{Hist\+Array}) and an associated observable function (O\+B\+S\+Fnc, see \hyperlink{Observables_8h}{Observables.\+h}). The phase space classes serve as the \textquotesingle{}source\textquotesingle{} of data for the distributions. The distribution class can thus be seen as the interface between these two. \begin{DoxySeeAlso}{See also}
