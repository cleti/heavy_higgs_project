\subsection{Detailed Description}
Class for differential distributions of mean values. 

The mean distribution is an extension of the distribution class. It stores a pointer to a second observable function (O\+B\+S\+Fnc). Not only the bare event weight is filled into a histogram, but it is multiplied by a second observable, thereby computing the mean value of the latter as a distribution in the first observable. \begin{DoxySeeAlso}{See also}
